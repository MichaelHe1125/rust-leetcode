# Rust Leetcode
练习使用Rust语言刷[leetcode](https://leetcode.com/problemset/all/)或者[力扣](https://leetcode-cn.com/problemset/all/)算法题目, 一些不支持Rust判题的会使用Python进行解决,并对解题思路进行简单分析,分类,及总结.

## Environment
- rustc 1.37.0
- cargo 1.37.0

## How to test
```
cargo test  // 测试所有的测试
cargo test p0007_reverse_integer  // 只测试单个模块下的测试
```
